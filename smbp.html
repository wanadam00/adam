Project Overview

Sistem Maklumat Bersepadu Pengakap Malaysia (SMBP) is an integrated information system designed for managing registrations and memberships for Pengakap Malaysia (Malaysian Scouts Association). I was responsible for implementing fixes and enhancements to improve system stability, user experience, and functionality. The system uses Laravel as the backend framework with custom CSS styling and is deployed on AWS infrastructure.

Key Features

User-Friendly Interface

The system features an intuitive and modern design that enhances user navigation for managing registrations and memberships efficiently.

Member Registration

Comprehensive registration system for new members with validation, document upload, and approval workflows.

Payment Gateway (SenangPay)

Integrated SenangPay payment gateway requiring users to complete payment before receiving their registration card, ensuring secure and streamlined financial transactions.

Membership Management

Tools for managing member profiles, renewals, transfers, and tracking membership status across different levels.

Reporting System

Generate comprehensive reports on membership statistics, event participation, and organizational performance.
