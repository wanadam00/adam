Project Overview

This work focuses on integrating IMF commodity index data into the client website gcsg-grosvenor.com. The implementation displays three core indices: Coal Index, Natural Gas Index, and Oil Index in chart format, with clean data normalization and predictable update behavior.

Key Features

IMF API Integration

Commodity index data is sourced from the IMF API and displayed on the client website in a clear chart-based interface.

Combined Chart View

A combined chart compares Oil, Natural Gas, and Coal indices in one view for quick performance comparison.

Separate Chart Views

Individual charts are available for each index to provide focused trend analysis without overlapping datasets.

Dropdown Time Filters

Users can filter chart data using dropdown options for Monthly, Quarterly, and Annual timeframes.
