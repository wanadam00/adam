Project Overview

Student Storage is a lightweight static website built to promote item storage services for students. The system provides a clean and responsive design to showcase services, pricing, process, and facility details. It includes integrations with WhatsApp API for direct messaging and Google Maps for real-time directions. The site is ideal for businesses that don’t require a backend but still want an effective web presence.

Key Features

Static Web Application

Built using only HTML, Tailwind CSS, and JavaScript — no backend or database involved, ensuring fast performance and easy deployment.

Service Highlights

Clearly structured sections explaining available storage services, pricing plans, process steps, and unique features like facility security.

Responsive Design

Fully responsive layout optimized for desktop, tablet, and mobile users using Tailwind CSS utility-first classes.

WhatsApp Integration

A direct link to WhatsApp chat using the WhatsApp API, allowing customers to instantly contact the service owner.

Google Maps Routing

Integrated Google Maps API link enables users to click and view navigation routes directly to the facility.

Lightweight Deployment

Easily deployed on platforms like GitHub Pages or Netlify since the system does not rely on server-side processing.
