Job Application Website - Opih

Overview:

Opih is a comprehensive job application website designed to connect job seekers with employers in a streamlined and efficient manner. The platform aims to simplify the job search process, making it easier for candidates to find suitable opportunities while allowing employers to discover and engage with top talent. With a focus on user experience and functionality, Opih serves as a valuable resource for both job seekers and recruiters.

Key Features:

  1. User-Friendly Interface:
    • The website features an intuitive and modern design that enhances user navigation. Job seekers can easily browse job listings, while employers can navigate the platform to post job openings and manage applications.
  2. Advanced Job Search:
    • Users can search for jobs using various filters, including job title, location, industry, and experience level. This functionality allows candidates to quickly find relevant positions that match their skills and preferences.
  3. Profile Creation and Management:
    • Job seekers can create personalized profiles to showcase their resumes, skills, and work experience. This feature allows them to apply for jobs quickly and enables employers to view candidate profiles easily.
  4. Application Tracking System:
    • Opih includes a robust application tracking system that allows job seekers to monitor the status of their applications. Employers can also manage and review applications efficiently, streamlining the hiring process.
  5. Employer Dashboard:
    • Employers have access to a dedicated dashboard where they can post job listings, review applications, and communicate with candidates. This centralized system enhances the recruitment experience and improves efficiency.
  6. Resume Builder:
    • The platform features a user-friendly resume builder that guides job seekers through the process of creating professional resumes. This tool helps candidates present their qualifications effectively.
  7. Notifications and Alerts:
    • Users receive notifications about new job postings, application updates, and messages from employers, ensuring they stay informed throughout the job application process.
  8. Mobile Optimization:
    • Opih is fully optimized for mobile devices, allowing users to search and apply for jobs on-the-go, making the job search process more accessible.

Technologies Used:

  • Frontend: HTML, CSS, JavaScript, Laravel Livewire framework
  • Backend: PHP
  • Database: MySQL
